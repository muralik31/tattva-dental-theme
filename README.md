## Customization

### Colors
The theme uses CSS variables for easy customization. Main colors defined in `style.css`:

```css
--color-primary: #40E0D0;      /* Cyan/Turquoise */
--color-dark: #000000;         /* Background */
--color-surface: #0a0a0a;      /* Card backgrounds */
```
